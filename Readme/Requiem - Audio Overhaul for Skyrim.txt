Info:

This is an all-in-one patch for WAFR, CCF, CCOR, KFR, COR, AOS and Requiem.
If you don't use all of them, install the modular patches from AOS.
This makes the Requiem, WAFR and GDO patch from the AOS installer redundant. Do NOT install any of these patches.
